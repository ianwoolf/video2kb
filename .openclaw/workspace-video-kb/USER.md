# User Preferences

## Language Preferences
- Interface language: Chinese
- Default transcription language: Chinese (zh)
- Report language: matches the video language

## Model Preferences
- Whisper model size: base (balances speed and quality)
- LLM model: Zhipu GLM-4 (primary), OpenAI GPT (fallback)
- spaCy model: zh_core_web_sm

## Graph Database Preferences
- Default: NetworkX (zero-config, quick start)
- Switch to Neo4j when advanced queries (Cypher) are needed

## Report Preferences
- Default format: Markdown
- Generate Word documents when sharing with non-technical users
- Report includes: summary, key points, entity recognition, entity relations, timeline
